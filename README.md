# Node Auth Sequelize API 🔐

Authentication & Role-based Authorization API built using Node.js, Express, Sequelize & MySQL.

## 🚀 Features
- JWT Authentication
- Role-based Authorization
- Company Management
- Sequelize Migrations
- Modular Folder Structure

## 🛠 Tech Stack
- Node.js
- Express.js
- Sequelize ORM
- MySQL
- JWT
